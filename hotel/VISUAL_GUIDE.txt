💡 TIPS & TRICKS
═══════════════════════════════════════════════════════════════════════

✓ Keep terminal windows open - they show helpful logs
✓ Allow Windows Firewall when prompted
✓ Backup weekly using BACKUP_DATABASE.bat
✓ Use MENU.bat if unsure what to do
✓ Check START_HERE.txt for getting started
✓ Desktop shortcuts make life easier
✓ CHECK_SYSTEM.bat diagnoses most issues
✓ Both servers must run for the system to work
✓ Backend (8000) must start before frontend (3000)
✓ Don't close terminal windows while using the system
