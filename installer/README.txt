====================================================
  LADAPALA RESTAURANT POS - INSTALLATION GUIDE
====================================================

BEFORE YOU START:
1. Make sure you have administrator privileges
2. Place Node.js and Python installers in this folder:
   - nodejs/node-v20.11.0-x64.msi
   - python/python-3.11.0-amd64.exe

INSTALLATION:
1. Right-click INSTALL.bat
2. Select "Run as administrator"
3. Follow the prompts
4. Wait for installation to complete

AFTER INSTALLATION:
- Run START.bat to start the application
- Run STOP.bat to stop the application
- Run STATUS.bat to check status
- Run LOGS.bat to view logs
